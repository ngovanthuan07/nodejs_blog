😀 Init project 
```
npm init
```
😀 Download Express: https://www.npmjs.com/package/express
```
npm install express
```
😂 Install nodemon & inspector: https://www.npmjs.com/package/nodemon
```
npm i nodemon --save-dev
```

😁 run project
```
node index.js
``` 
nếu có cấu hình trong file package.json:
```
npm start
```
